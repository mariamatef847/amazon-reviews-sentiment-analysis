## Overview
This project performs *Sentiment Analysis* on the Amazon Food Reviews dataset using Python.  
It involves:
- Text preprocessing and cleaning using NLTK  
- Sentiment classification using TextBlob  
- Data visualization with Matplotlib and Seaborn  
- WordClouds for most frequent words in positive and negative reviews  

## Project Steps
1. *Load Data* ‚Äì Import Amazon reviews dataset.  
2. *Text Cleaning* ‚Äì Remove HTML tags, punctuation, and stopwords.  
3. *Sentiment Analysis* ‚Äì Use TextBlob to classify each review.  
4. *Visualization* ‚Äì Create a bar chart for sentiment distribution and a WordCloud for most frequent words.
